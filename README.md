# 贝壳校园物语

🎮 人工智能大作业：2D像素校园养成游戏

---

## 游戏简介

**贝壳校园物语**是一款以北京科技大学为背景的校园养成游戏。玩家将扮演一名大学新生，在四年的大学生活中，通过学习、运动、参加社团等活动，提升各项能力，度过精彩的大学时光，最终达成不同的结局。

---

## 快速开始

### 方式一：直接运行（推荐）

双击运行 `dist/贝壳校园物语.exe` 即可启动游戏。

### 方式二：源码运行

**环境要求**
- Python 3.8+
- Pygame 库

**安装依赖**
```bash
pip install pygame
```

**运行游戏**
```bash
python main.py
```

---

## 游戏操作

### 基本操作
- **鼠标左键点击**：选择/确认
- **键盘快捷键**：
  - `ESC`：返回主界面

### 界面导航
- 点击底部功能图标可快速打开对应界面
- 点击地图可切换不同场景

## 游戏系统

### 属性系统

玩家拥有三大类属性：

| 属性类型 | 属性名称 | 说明 |
| :--- | :--- | :--- |
| **主属性** | 学识、魅力、体能 | 最高4级，影响期末成绩和结局 |
| **生活属性** | 金钱、行动力、心情、健康 | 日常活动消耗，影响游戏体验 |
| **副属性** | 技能、人脉、声望 | 辅助属性，解锁特殊事件 |

### 背包系统
- 背包容量：20格
- 在超市购买物品后，物品会存入背包
- 使用物品会消耗1个并获得对应效果

### 超市物品

| 物品名称 | 价格 | 效果 |
| :--- | :--- | :--- |
| 美味蛋糕 | 15 | 心情+30，人脉+2 |
| 潮流衣服 | 30 | 魅力+10，声望+2 |
| 课外教材 | 20 | 学识+10，技能+3 |
| 健身器材 | 30 | 体能+10，声望+2 |
| 不健康的零食 | 15 | 心情+40，健康-5，人脉+1 |
| 体力药水 | 30 | 行动点+1 |

### 地图地点

| 地点 | 功能 |
| :--- | :--- |
| 🏠 宿舍 | 休息恢复行动力 |
| 🍽️ 食堂 | 用餐恢复体力 |
| 🏫 教学区 | 上课提升学识属性 |
| ⚽ 操场 | 运动提升体能 |
| 🛒 超市 | 购买物品存入背包 |
| 🎭 学生活动中心 | 社交活动提升人脉 |
| 🏥 校医院 | 治疗疾病恢复健康 |

## 游戏目标

在128天的校园生活中，努力提升各项属性，根据综合表现达成不同结局：

- 🏆 **保研** - 优异成绩获得保研资格
- 📚 **考研上岸** - 通过考研继续深造
- 💼 **优质就业** - 获得优质工作offer
- 🎯 **普通就业** - 顺利毕业就业
- 🚀 **创业** - 自主创业开启新篇章
- ⚠️ **延毕** - 未达到毕业要求

---

## 游戏提示

1. 📅 合理安排日程，平衡学习与生活
2. ❤️ 注意保持健康值高于40，否则会生病
3. 😊 心情低于50时，属性效果会减半
4. 📝 每学期末会有期末考试，成绩影响结局
5. 💾 记得定期存档，避免丢失进度
